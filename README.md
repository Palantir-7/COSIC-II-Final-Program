# COSIC-II-Final-Program
This code takes a string of characters and converts it to ciphers.
The program has three different options currently for encoding messages;
it has a Ceasar Cipher function, a Vingere Cipher function, and a function that simply
converts the string of letters into their corresponding numeric values, for example: a = 0...
The Vingere and Ceasar Ciphers are substitution ciphers, the only difference is one takes
a keyword to encrypt it and the other simply takes a number to shift it.
The output for the str2num function is wonky and I have not figured it out yet, but
the actual encryption is working though.
